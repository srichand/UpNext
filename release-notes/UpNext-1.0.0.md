# UpNext 1.0.0

The first public release of UpNext, a lightweight macOS menu bar companion for your calendar.

## Highlights

- See your next meeting and a live countdown directly in the menu bar.
- Browse meetings for previous and upcoming days.
- Choose which calendars UpNext monitors.
- Start UpNext automatically when you log in.
- Install secure, signed updates automatically with Sparkle.
- Install and update UpNext through the `srichand/tap` Homebrew tap.

UpNext keeps calendar data on your Mac and does not include analytics or tracking.
