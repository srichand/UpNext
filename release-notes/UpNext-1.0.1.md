# UpNext 1.0.1

This update gives UpNext a more polished, unmistakably native Mac interface and makes background updates easier to notice.

## Improvements

- Refined the popover's typography, spacing, event rows, and navigation controls.
- Improved countdown contrast and added clearer states for imminent and active meetings.
- Replaced the ambiguous "Today" navigation action with "Go to Today."
- Reorganized Settings with General first and added standard Command–Comma support.
- Added a gentle menu-bar and popover reminder when a Sparkle update is available.
- Preserved native keyboard focus while removing the misleading initial selection appearance.

## Requirements

UpNext now requires macOS 15 Sequoia or later. Macs running macOS 14 can continue using UpNext 1.0.0.
