# UpNext Privacy Policy

Last updated: July 9, 2026

UpNext is designed to keep your calendar information private.

## Calendar data

UpNext requests read access to your calendars so it can show upcoming events in the macOS menu bar. Calendar events are read and processed on your Mac. UpNext does not transmit calendar titles, attendees, locations, notes, or other event details to Srichand Pendyala or any third party.

Your selected calendar identifiers and app preferences are stored locally using macOS preferences.

## Network access

UpNext uses network access only to check GitHub for signed software updates through the Sparkle update framework. GitHub may receive standard connection information such as your IP address and user agent under GitHub's own privacy policy.

UpNext does not include analytics, advertising, tracking, crash-reporting, or telemetry services.
